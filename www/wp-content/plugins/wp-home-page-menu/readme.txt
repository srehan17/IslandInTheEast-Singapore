This plugin displays home page menu in the navigation bar which can be configured easily from the admin area.

== Description ==

WP Home Page Menu plugin adds home page menu in your site navigation bar which points to the home page of your site.

The plugin provides you various settings in the admin area which can be used to change home page menu label, title and add classes to menu.

Also you can enable or disable home page menu from the plugin settings and control on which navigation menu on your site it should be displayed.

== Installation ==

* Install the plugin from the 'Plugins' section in your dashboard (Go to `Plugins -> Add New -> Search` and search for WP Home Page Menu).
* Alternatively, you can [download](http://downloads.wordpress.org/plugin/wp-home-page-menu.zip "Download WP Home Page Menu") the plugin from the repository. Unzip it and upload it to the plugins folder of your WordPress installation (`wp-content/plugins/` directory of your WordPress installation).
* Activate it through the 'Plugins' section.
* Use the `Settings -> WP Home Page Menu` to configure the plugin

== Frequently Asked Questions ==

= Why does the home page menu not display on my site? =

Please go to the Settings -> WP Home Page Menu in the admin area of your site and make sure that the plugin is enabled and correct menu locations
are selected. Also try checking the option "My Site Uses Pages Menu?".

If you already have configured correct plugin settings then just resave the settings.
